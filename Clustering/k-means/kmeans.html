Tal y como pide el enunciado, vamos a aplicar el algoritmo kmeans para k=2. Previamente, como el algoritmo hace uso de nÃºmeros aleatorios, vamos a hacer reproducible la ejecuciÃ³n estableciendo una semilla concreta:

k<-2
set.seed(1)
two.means <- kmeans(unempstates, k, iter.max = 10, nstart = 1)

Como la estrucutura de datos resultante de aplicar el algoritmo no deja extraer el grupo que se la ha asignado a cada estado, vamos a crear una funciÃ³n que se encargue de ello (hago esto porque si usamos directamente el campo clusters del objeto que devuelve la funciÃ³n, vamos a tener repetido dos veces el identificador del estado):

getClusterNumber <- function(cluster){
    v <- vector(mode = "numeric", length = 0)
    for(i in 1:length(cluster)){
      v <- c(v,cluster[[i]])
    }
    v
}

two.dataframe <- data.frame(state = names(two.means$cluster), cluster = getClusterNumber(two.means$cluster))
